# **App Name**: Revio Research Hub

## Core Features:

- Smooth Scrolling Navbar: A fixed navbar with smooth scrolling to different sections of the single-page application.
- Dynamic Section Reveal: Sections of the page dynamically open and close with smooth accordion and reveal animations.
- Contact Form Submission: Submit contact form data to Firestore, triggering smooth form validation animations.
- Testimonial Approval Tool: Admin tool to approve testimonials which are then displayed in the testimonials carousel. The tool reasons when to include the testimonial.
- User Authentication: Email & Password and Google Sign-In authentication with role-based access control stored in Firestore.
- Service Request Management: Authenticated users can create and view their own service requests, managed within Firestore.
- Firestore Security Rules: Secure Firestore data with production-ready security rules to control data access based on user roles and authentication status.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and intellect.
- Background color: Very light gray (#F5F5F5), almost white, to maintain a clean and spacious feel.
- Accent color: Muted Teal (#4DB6AC) for subtle highlights and calls to action, adding a touch of sophistication.
- Body and headline font: 'Inter' for a modern, neutral, and readable style.
- Card-based sections with soft, long shadows to subtly lift elements off the page.
- Underline animation on the active navbar tab for clear visual feedback.
- Subtle hover lift and shadow depth animation on cards for enhanced interactivity.
